プライバシーポリシー

1. 取得する情報

  • メールアドレス、表示名（アカウント登録時）
  • 位置情報（現在地検索を選択した場合のみ。サーバーには検索条件のみ保存）
  • 検索履歴、お気に入り、来店記録、評価、コメントなどのサービス利用情報
  • コメント投稿時の IP アドレス・User-Agent（スパム / 不正対策のため）
  • Cookie / セッション識別子（ログイン状態の維持・CSRF 対策）

2. 利用目的

  • 本サービスの提供、機能改善、レコメンドの精度向上
  • 不正利用・スパム・誹謗中傷の検出と対応
  • 統計データの作成（個人を特定しない集計）

3. 第三者提供

法令に基づく場合を除き、本人の同意なく個人情報を第三者に提供しません。検索のため、入力されたキーワードや位置情報は HotPepper API・Google Maps API・Google Gemini API に送信される場合がありますが、これらは個人を特定する形では送信されません。

4. データの保存期間

  • 検索履歴: 最終アクセスから 180 日
  • 店舗表示履歴: 最終アクセスから 90 日
  • アカウント情報: 退会後すみやかに削除（関連レコードもカスケード削除）

5. 利用者の権利

利用者は、自身のアカウント情報の確認・修正・削除（退会）をマイページから行えます。その他のお問い合わせはお問い合わせフォームをご利用ください。

6. Cookie

ログイン状態の維持と CSRF 対策のため、必須 Cookie を使用しています。広告目的の Cookie は使用していません。

7. 改定

本ポリシーは法令や運用状況に応じて改定する場合があります。重要な変更がある場合はサービス上で告知します。
